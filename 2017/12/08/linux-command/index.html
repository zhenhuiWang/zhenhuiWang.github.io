Linux commands

Posted on 2017-12-08 | In 技术 |

记得以前上学的时候学习这些命令真的记不住，因为老师把这些命令一列，然后自己就死记硬背，这种方法显然是不科学的。我们必须要借助英语的力量，因为这些命令基本上都是英语单词的缩写。

总结一些常用的命令以及他们的全称和缩写。

Command Description
ls [options] List directory contents.
cd /directorypath change directory
cat [filename] Display file’s contents to the standard output device
mkdir [options] directory make directory
rmdir [options] directory remove directory
rm [options] directory Remove (delete) file(s) and/or directories.
touch filename Create an empty file with the specified name.
pwd Print Working Directory
mv [options] source destination Rename or move file(s) or directories
cp [options] source destination Copy files and directories.
clear Clear a command line screen/window for a fresh start.
chmod [options] mode filename Change Mode, Change a file’s permissions.

所以英语对我们的编程学习有很大的帮助哦，如果想熟练掌握的话还得多用，每个命令都可以试着执行一遍，无他，唯手熟尔。
